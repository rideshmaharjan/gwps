# GWPS - Gym/Workout Package System

A full-stack web application for managing workout packages. Built as a 4th semester BCA project.

 Features
- User authentication (login/register)
- Admin dashboard
- Member management (add, edit, delete)
- Package management (workout plans, pricing)
- CRUD operations

Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML/CSS
- Version Control:Git & GitHub

## Project Status
Completed as part of BCA curriculum. Co-developed with a project partner.

How to Run Locally
1. Clone the repository
2. Import the database file
3. Configure `config.php` with your database credentials
4. Run on local server (XAMPP/WAMP)
